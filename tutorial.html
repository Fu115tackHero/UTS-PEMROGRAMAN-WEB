Penjelasan Mendetail Struktur HTML:

1. Deklarasi DOCTYPE dan Elemen HTML

<!DOCTYPE html> - Menentukan bahwa dokumen ini menggunakan HTML5.

<html lang="id"> - Elemen root dengan atribut bahasa Indonesia.

2. Bagian Head (Informasi Meta)

<meta charset="UTF-8"> - Menentukan encoding karakter untuk mendukung huruf internasional.

<meta name="viewport"> - Membuat halaman responsif di perangkat mobile.

<title> - Judul yang muncul di tab browser.

<link rel="stylesheet"> - Menghubungkan file CSS eksternal.

3. Bagian Body (Konten yang Ditampilkan)

<button id="darkModeToggle"> - Tombol untuk mengganti tema gelap/terang.

<div class="card"> - Container utama untuk kartu biodata.

<div class="biodata"> - Bagian untuk foto dan informasi pribadi.

<img> - Elemen untuk menampilkan foto profil.

<h1>, <h3>, <h4> - Heading dengan tingkat kepentingan berbeda.

<ul> dan <li> - List tidak terurut untuk menampilkan hobi.

<table> - Membuat tabel untuk jadwal kuliah dengan <thead> (header) dan <tbody> (body).

<tr> - Baris tabel, <th> - sel header, <td> - sel data.

4. Class dan ID yang Digunakan

class="card" - Styling untuk kartu biodata.

class="biodata" - Mengatur layout foto dan info pribadi.

class="image" - Styling khusus untuk foto profil.

id="darkModeToggle" - Identifikasi unik untuk tombol dark mode.

class="social-links" - Container untuk link media sosial.

Penjelasan Mendetail CSS:

1. Variabel CSS (:root)

Mendefinisikan variabel warna untuk konsistensi dan kemudahan perubahan tema.

--primary-color - Warna utama (biru), --secondary-color - Warna sekunder (biru tua)

--text-light/dark - Warna teks untuk tema terang/gelap

--bg-light/dark - Warna background untuk tema terang/gelap

2. Reset dan Style Dasar

* { margin: 0; padding: 0; box-sizing: border-box; } - Reset default browser dan membuat perhitungan ukuran lebih mudah.

body - Mengatur font, background, layout flexbox, dan transisi untuk perubahan tema.

3. Styling Komponen

#darkModeToggle - Tombol dengan posisi fixed, efek hover, dan transisi.

.card, .jadwal-kuliah - Container dengan background, shadow, border-radius, dan efek hover.

.biodata - Menggunakan flexbox untuk layout horizontal foto dan info.

.image - Foto profil dengan bentuk bulat, border, dan efek hover scale.

.social-links - Container flexbox untuk ikon media sosial.

.social-link - Ikon bulat dengan transisi dan efek hover.

table - Tabel dengan border-collapse, shadow, dan styling sel.

tr:nth-child(even) - Warna alternating untuk baris tabel.

4. Dark Mode Styles

body.dark-mode - Mengubah warna background dan teks saat dark mode aktif.

Semua class dengan prefix .dark-mode mengoverride nilai variabel CSS untuk tema gelap.

5. Media Queries (Responsive Design)

@media (max-width: 768px) - Style untuk tablet dan layar kecil.

@media (max-width: 480px) - Style untuk mobile phone.

Mengubah layout biodata menjadi vertikal, menyesuaikan ukuran font, dan padding.

Langkah 4: Menambahkan Interaktivitas dengan JavaScript

JavaScript membuat website kita interaktif. Di langkah ini, kita akan menambahkan fungsionalitas toggle dark mode yang dapat diaktifkan oleh pengguna.

File: script.js

Salin kode JavaScript berikut ke dalam file script.js: 

// Menunggu hingga seluruh konten HTML selesai dimuat
document.addEventListener('DOMContentLoaded', function() {
    // Mendapatkan referensi ke tombol toggle dan body
    const darkModeToggle = document.getElementById('darkModeToggle');
    const body = document.body;
    
    // Memeriksa apakah pengguna sebelumnya memilih dark mode
    const savedTheme = localStorage.getItem('theme');
    
    // Jika ada tema yang disimpan, terapkan tema tersebut
    if (savedTheme === 'dark') {
        body.classList.add('dark-mode');
        darkModeToggle.textContent = 'â˜€ï¸ Light Mode';
    } else {
        darkModeToggle.textContent = 'ðŸŒ™ Dark Mode';
    }
    
    // Menambahkan event listener untuk klik tombol
    darkModeToggle.addEventListener('click', function() {
        // Toggle class 'dark-mode' pada body
        body.classList.toggle('dark-mode');
        
        // Memeriksa apakah dark mode aktif
        const isDarkMode = body.classList.contains('dark-mode');
        
        // Mengubah teks tombol berdasarkan status dark mode
        if (isDarkMode) {
            darkModeToggle.textContent = 'â˜€ï¸ Light Mode';
            // Menyimpan preferensi tema ke localStorage
            localStorage.setItem('theme', 'dark');
        } else {
            darkModeToggle.textContent = 'ðŸŒ™ Dark Mode';
            // Menyimpan preferensi tema ke localStorage
            localStorage.setItem('theme', 'light');
        }
        
        // Menambahkan efek feedback visual
        darkModeToggle.style.transform = 'scale(0.95)';
        setTimeout(() => {
            darkModeToggle.style.transform = 'scale(1)';
        }, 150);
    });
    
    // Menambahkan animasi masuk untuk elemen kartu
    const cards = document.querySelectorAll('.card, .jadwal-kuliah');
    cards.forEach((card, index) => {
        card.style.opacity = '0';
        card.style.transform = 'translateY(20px)';
        
        setTimeout(() => {
            card.style.transition = 'opacity 0.5s ease, transform 0.5s ease';
            card.style.opacity = '1';
            card.style.transform = 'translateY(0)';
        }, index * 200);
    });
    
    // Menambahkan efek hover yang lebih smooth untuk social links
    const socialLinks = document.querySelectorAll('.social-link');
    socialLinks.forEach(link => {
        link.addEventListener('mouseenter', function() {
            this.style.transition = 'all 0.3s ease';
        });
        
        link.addEventListener('mouseleave', function() {
            this.style.transition = 'all 0.3s ease';
        });
    });
});

// Fungsi untuk menampilkan notifikasi (opsional)
function showNotification(message, type = 'info') {
    const notification = document.createElement('div');
    notification.textContent = message;
    notification.style.cssText = `
        position: fixed;
        top: 20px;
        left: 50%;
        transform: translateX(-50%);
        padding: 12px 20px;
        border-radius: 8px;
        color: white;
        font-weight: 500;
        z-index: 1001;
        transition: all 0.3s ease;
    `;
    
    if (type === 'success') {
        notification.style.backgroundColor = '#48bb78';
    } else if (type === 'error') {
        notification.style.backgroundColor = '#f56565';
    } else {
        notification.style.backgroundColor = '#4299e1';
    }
    
    document.body.appendChild(notification);
    
    // Menghilangkan notifikasi setelah 3 detik
    setTimeout(() => {
        notification.style.opacity = '0';
        notification.style.transform = 'translateX(-50%) translateY(-20px)';
        setTimeout(() => {
            document.body.removeChild(notification);
        }, 300);
    }, 3000);
}

Penjelasan Mendetail JavaScript:

1. Event DOMContentLoaded

document.addEventListener('DOMContentLoaded', function() { ... })

Menunggu sampai seluruh dokumen HTML selesai dimuat sebelum menjalankan kode JavaScript. Ini penting karena kita perlu memastikan elemen yang akan dimanipulasi sudah tersedia.

2. Mendapatkan Elemen DOM

const darkModeToggle = document.getElementById('darkModeToggle') - Mendapatkan tombol dark mode berdasarkan ID.

const body = document.body - Mendapatkan referensi ke elemen body.

3. Mengelola Preferensi Tema

const savedTheme = localStorage.getItem('theme') - Mengecek apakah pengguna sudah memilih tema sebelumnya.

localStorage adalah API browser untuk menyimpan data secara persisten.

Jika tema 'dark' tersimpan, tambahkan class 'dark-mode' ke body dan ubah teks tombol.

4. Event Listener untuk Tombol

darkModeToggle.addEventListener('click', function() { ... }) - Menambahkan pendengar event klik ke tombol.

body.classList.toggle('dark-mode') - Menambah atau menghapus class 'dark-mode' dari body.

classList.contains('dark-mode') - Mengecek apakah class 'dark-mode' ada di body.

Berdasarkan status dark mode, ubah teks tombol dan simpan preferensi ke localStorage.

5. Efek Visual dan Animasi

darkModeToggle.style.transform = 'scale(0.95)' - Efek tekan saat tombol diklik.

setTimeout() - Mengembalikan ukuran tombol setelah 150ms.

document.querySelectorAll('.card, .jadwal-kuliah') - Memilih semua elemen kartu dan jadwal.

Animasi masuk: Setiap kartu muncul dengan delay yang berbeda (index * 200ms).

6. Logika Fungsi showNotification (Opsional)

Membuat elemen div baru untuk notifikasi.

document.createElement('div') - Membuat elemen HTML baru.

document.body.appendChild(notification) - Menambahkan notifikasi ke body.

setTimeout() - Menghapus notifikasi setelah 3 detik dengan efek fade out.

7. Alur Kerja Dark Mode
  1. User mengklik tombol dark mode
  2. JavaScript menambah/hapus class 'dark-mode' dari body
  3. CSS dengan selector body.dark-mode mengaktifkan styling tema gelap
  4. Preferensi tema disimpan di localStorage
  5. Saat halaman dimuat ulang, JavaScript membaca localStorage dan menerapkan tema yang disimpan

Langkah 5: Testing dan Menjalankan Website

Cara Menjalankan Website:

  1. Buka File HTML: Buka folder proyek Anda dan klik dua kali pada file index.html untuk membukanya di browser default
  2. Testing di VS Code: Jika menggunakan VS Code, Anda dapat menginstal ekstensi Live Server lalu klik kanan pada file HTML dan pilih "Open with Live Server" untuk melihat perubahan secara real-time
  3. Testing Fitur:
    • Coba klik tombol "Dark Mode" untuk mengubah tema
    • Refresh halaman dan periksa apakah preferensi tema tersimpan
    • Ubah ukuran jendela browser untuk melihat responsivitas
    • Coba akses dari perangkat mobile atau gunakan dev tools browser
